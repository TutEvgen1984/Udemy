10 способов выравнивания элемента по вертикали и горизонтали

Код автора на gitHub

Метод 1


        .parent1 {
            position: relative;
        }
        .child1 {
            position: absolute;

            /* Смещение верхнего края на 50% вниз - 50% от высоты родителя */
            top: 50%;

            /* Смещение левого края на 50% вправо - 50% от ширины родителя */
            left: 50%;

            /* В результате левый верхний угол центрируемого оранжевого квадрата оказался точно посередине родительского квадрата */

            transform: translate(-50%, -50%);
            /* -50% по горизонтали и -50% по вертикали */
            /*  отличии от top и left translate двигает элемент не относительно левого верхнего края, а относительно его центра */
            /*  в результате мы вернули его обратно на -50% относительно своего размера влево и вверх  благодаря отрицательным значениям -50% */

        }
