Способ 2. text-align: center

CSS:
        .container1 {
            /* центрирование div'а */
            margin: 0 auto;

            /* центрирование картинки в div'е как текста, так как она является строчным элементом изначально */
            text-align: center;

            background-color: tomato;
            width: 500px;
        }

Но внизу остался небольшой зазор так как img строчный. Высота картинки не равна высоте блока

Способ 3. display flex + justify-content center

        .container2 {
            /* делаем контейнер флексом */
            display: flex;

            /* text-align: center; */
            /* центрирование текстом теперь не работает так как это теперь флекс-блок */

            /*Для флекс-блоков выравнивание теперь работает другим свойством justify-content */
            justify-content: center;

            background-color: tomato;
            width: 500px;

            /* центрирование div'а */
            margin: 0 auto;

        }

В данном случае зазора нет в отличии от предыдущего способа. Высота картинки равна высоте блока

Как выровнять содержимое строчного элемента по центру?

Ответ: text-align: center;

2 изображения вместо 1;

Но есть особенность - 2 изображения ищущие друг за другом будут в одну сторону, а не друг под другом как в блочной модели

Чтобы это изменить делаем следующее

CSS:
        .container4 {
            display: flex;

            /* Направление flex делаем колонкой */
            flex-direction: column;

            /* justify-content теперь перестает работать */
            /* justify-content: center; */

            background-color: tomato;
            width: 1000px;

            margin: 0 auto;

        }

        /* Изображению указываем свойство самовыравнивания: */
        .img4 {
            align-self: center;
        }

Теперь изображения не растягиваются и идут как в блочной верстке друг под другом, но при этом выравниваются по горизонтали. Этот метод не стоит применять когда работают первые 2 способа

Как исправить баг, при котором изображение искажается при работе с Flexbox?

Ответ: align-self: center;
