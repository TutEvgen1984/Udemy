Важные сочетания клавиш Visual Studio Code:

  1. Закомментировать выделенный текст: ALT+SHIFT+A.
  2. Закомментировать/раскоментить текущую строку на которой установлен курсор: CTRL+/(рядом с правым шифтом).
  3. Создать несколько курсоров: ALT+Shift (зажать) на клаве и щелкнуть и тянуть ЛК мышки вверх/вниз. В результате создаются несколько курсоров в нескольких строках чтобы редактировать текст одновременно вертикальным блоком. В добавок так можно вставить из буфера обмена слова в несколько строк одновременно. Зажать ALT и щелкнуть ЛК мышки в любое место и курсоров станет несколкьо - и можно писать в нескольких
  4. Копирование строки кода VS Code: ALT+SHIFT+↓ нажать находясь в VS Code на строке кода, в результате текущая строка кода скопируется вниз.
  5. Копирование строк списка (для li): ALT+SHIFT+↓. Кодируя список находясь строке li нажать сочетание, в результате внизу появится дополнительная строка и туда скопируется исходная строка (это работает если в строке был набран простой текст без всяких хитрений и переносов, ещё лучше когда в строке вообще ничего не набрано)
  6. Перенос по словам в Studio Code: ALT + Z.
    Если вместо него срабатывает оверлей GEFORCE, то правой кнопкой на geforce experience -> Общие -> Внутриигровой оверлей -> выключть пипку.
  7. lorem ipsum (текст-рыба): набрать Lorem и нажать ENTER - в результате сгененрируется текст на латыни (по умолчанию текст из 30 слов).
    Если набрать lorem5 и затем интер - сгененрируется текст из 5 слов.
  8. Показать доступные варианты кода: CTRL+Space.
  9. Автозаполнение кода: ENTER, либо TAB, либо ЛКМ. Набрать любой тег, например p, и нажать клавиши - в результате Studio Code поставит все нужные скобки и закрет тег.
    Например набираем script, видим 3 варианта, нажимаем стрелку низ 2 раза, нажимаем интер либо таб, получаем головый код скрипта <script src=""></script>.
