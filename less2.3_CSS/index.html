Quiz 3. Вопросы к главе по языку CSS

Question 1:

Какой селектор позволяет выбрать все изображения с атрибутом alt?



Ответ: [alt]

Question 2: Как вы пути файла указать выход на один уровень выше

Ответ: ../

Question 3: Какие параметры есть у свойства border для создания границ

Отве: color, width, style

Question 4: Какие свойства позволяют изменять ширину и высоту элемента

Ответ: width, height

Question 5: Как указать перенос элементов на новую строку

Ответ: flex-wrap

Question 6: С помощью какой конструкции создается адаптивный дизайн

Ответ: @media
