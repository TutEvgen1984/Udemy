Question 1: Как с помощью CSS задать отступ шириной в 5px внутри элемента?

Ответ: padding 5px

Ответ: @font-face. https://webref.ru/css/font-face

Question 3: Изменение какого свойства возможно сделать плавным с помощью transition?

Ответ: opacity

Question 4: Возможно ли установить больше одной картинки на фон элемента с помощью CSS?

Ответ: да. https://snipp.ru/html-css/multiple-backgrounds

Question 5: Rаково изначальное значение CSS свойства background-origin?

Ответ: padding-box. https://webref.ru/css/background-origin

Question 6: Выберете некорректное значение свойства transform:

Ответ: rotate(30degrees) неправильная запись, должно быть rotate(30deg). scale(200%) правильно, rotate(0.5turn) правильно, matrix(1,2,3,4,5,6) правильно. https://webref.ru/css/transform, https://webref.ru/css/value/rotate

Question 7: Как размыть изображение?

Ответ: filter: blur(4px). https://webref.ru/css/value/blur

Question 12: Возможно ли указать font-weight, font-style и font-size, используя только одно CSS свойство?
