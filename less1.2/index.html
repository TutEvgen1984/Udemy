Таблицы

table (от англ. table — таблица) служит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек, которые задаются с помощью элементов tr (от англ. table row — строка таблицы) и td (от англ. table data cell — ячейка с данными таблицы). Внутри table допустимо использовать следующие элементы: caption, col, colgroup, tbody, td, tfoot, th, thead и tr.

Элемент div
Элемент div (от англ. division — раздел, секция) является универсальным блочным элементом и предназначен для группирования элементов документа с целью изменения вида содержимого через стили. Для этого добавляется атрибут class или id с именем класса или идентификатора. Как и при использовании других блочных элементов, содержимое div всегда начинается с новой строки, после него также добавляется перенос строки.
